Let's look at the relationship between Javascript, HTML and CSS

Using the analogy of a pop-up book:

HTML: Would represent the words on the page, the contents written in the book and the structure.

CSS: Would represent the font and styling of the words on the page, how the words are presented in terms of font sizing, colour, spacing and font colour.

Explain control flow and loops using an example process from everyday

Control Flow:

Control flow describes the way code is executed depending on the order that it is read. Since code is read in the order of beginning to end from the way it is presented in a program, that will be the order in which the code is executed.

Let’s imagine that you are working in a bakery where you sell muffins. On your first day of work your boss tells you that when the cabinet is down to 3 muffins left, you must follow the three steps of baking more muffins, frosting them and stocking up the cabinet. These steps are done in sequential order where you check whether the baked goods cabinet is down to 3 muffins, if it is then you go and bake more muffins, frost them then stock up the cabinet. These steps are what represent a control flow as they are a process that follows a specific sequence.

Loops:

Loops are used for situations where you may do the same task the exact same way again without having to repeat the same instruction.

Now, because you’re a smart employee that knows how to loop, instead of the boss telling you all the steps of checking the baked goods cabinet, baking, frosting, and stocking up, every time you see that there are only 3 muffins left in the baked good cabinet you repeat the boss’ 3 instructions.

Here, you are demonstrating a loop in action, where the loop (following the boss’ three instructions) is initiated when the condition of 3 muffins left in the cabinet is met.

As soon as you stock up the cabinet and there are more than three muffins, the condition is no longer met and so the loop stops and you do not bake more muffins.

Therefore, the process of a loop demonstrates efficiency as you automate a set of instructions and then use a loop to repeat them until specific conditions are met.

Explain the difference between accessing data from arrays and objects.

Array:

Arrays are laid out in an ordered list format where each element in the array has an index, basically like a number associated with it to determine its position in the array.

Just like if you were to line up at the supermarket and there are other people behind you, if you are at the front you will be given the number 1 and the person behind you would be 2 and the one behind them 3 and so on.

However, the thing to note about counting indexes is that it always starts from number 0. So if we use the supermarket analogy again, you would not be number 1 but number 0 and the person behind you would be number 1.

Arrays are usually used when ordering a list that is in the same category, an example would be to use an array for items you want to buy at the supermarket.

To access an element within the array, you use the numeric index as seen in the example below.



Object:

Objects are used to organise based on categories and use a key-value pair. The key is a string and the value in the key can be any data type (string, number). To access a value in the object, you use the key that the value is under.

There are two ways to access values in an object, square bracket notation and dot notation. Using square brackets will give you all of the values under the key as one string (e.g. artistAlbums[Charlie])

Whereas dot notation is more concise and will give you the specific value you are looking for under the key (e.g. artistAlbums.artist).


Explain what functions are and why they are helpful.

Functions are used to perform one specific task or multiple different tasks. Functions are given a set of code instructions and then when the function is called, it will be activated and run through the set of code instructions to perform the task.

Functions can be used to make coding more efficient and organised as you can write one function and have it be used multiple times with different parameters without having to rewrite the task over and over again.
